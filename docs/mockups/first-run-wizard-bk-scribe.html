Настройка BK Scribe

Подготовьте приложение к записи, транскрипции и формированию итогов встреч.

Настройка не завершена Рабочий день станет доступен после всех обязательных проверок.
Пройдите шаги настройки. Рабочий день откроется после успешной проверки OBS, аудио, AI Tunnel, транскрипции и AI-итогов.
2 из 7 готово

Папка данных

Выберите место, где BK Scribe будет хранить рабочие дни, записи, транскрипты и итоги.

Готово
Данные сохраняются отдельно от приложения

При обновлении или удалении BK Scribe рабочие дни останутся в выбранной папке.

Запись разговоров

BK Scribe записывает встречи через OBS. Откройте OBS и убедитесь, что WebSocket включен.

Требует действия
OBS не подключен

Запустите OBS, проверьте, что WebSocket включен, затем нажмите проверку. Без OBS рабочий день будет недоступен.

Аудио из записей

Проверяем, что BK Scribe сможет извлекать аудио из записей после завершения встречи.

Готово
Аудио из записей готово

Встроенный FFmpeg найден. Дополнительная установка пользователю не нужна.

AI Tunnel

Введите один ключ AI Tunnel. Он будет использоваться и для транскрипции, и для формирования итогов.

Не проверено
Ключ сохранится только локально в файле `.env` на этом компьютере.
Где получить ключ

Если ключа еще нет, оформите его на сайте aitunnel.ru, затем вернитесь в BK Scribe и вставьте ключ сюда.

Транскрипция

Выберите способ распознавания речи. Для коллег транскрипция обязательна.

Не проверено
AI Tunnel выбран по умолчанию как основной режим для коллег.
Проверка без реальной встречи

Для AI Tunnel STT используется ключ из предыдущего шага. Мастер проверит настройки транскрипции без создания записи, transcript или рабочего дня.

AI-итоги

Настройте модель, которая будет формировать итоги встреч и дня. Это обязательный шаг.

Не проверено
Эта модель будет формировать итоги встреч и итог дня.
Тест коротким текстом

Для итогов используется тот же ключ AI Tunnel. BK Scribe отправит небольшой тестовый текст в выбранную модель. Реальные записи, transcript и итоги пользователя не создаются.
