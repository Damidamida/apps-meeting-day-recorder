# Agent Guidance

## Project goal

Build a local-first Windows desktop app for manually recording ad hoc workday meetings and reviewing generated meeting artifacts.

## Constraints

- Keep meeting data local-first.
- Do not add email or calendar integration in the MVP.
- Treat manual meeting start and stop controls as the main MVP flow.
- Do not store secrets or recordings in git.
- Always work in branches.
- Do not push directly to `main`.
- Keep changes small and reviewable.
